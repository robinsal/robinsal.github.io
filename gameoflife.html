Game of Life

This is a simple JavaScript program that alters
the DOM Tree and creates allows you to play
the Game of Life. Enjoy!


Rules of the game:
Cells are either 'alive' or 'dead'.
Live cells are black, dead cells white.
A live cell will stay alive if it has 2 or 3
live neighbors, and will die otherwise.
A dead cell will come to life if it has exactly 3 live neighbors.


This game was created by John Conway, and it
produces some pretty cool visuals. I recommend
you begin by clicking Set Random Board to start out,
and then you can set any cell to be alive by clicking on
it. Update the game by clicking Update. To create interesting
visuals, click the Update button rapidly.
